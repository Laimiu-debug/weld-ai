### 阶段2 关键核实结论

1. **不锈钢分类澄清**（消除阶段0冲突）：
   - **Fe-6** = 马氏体不锈钢（不分组）：06Cr13/12Cr13/20Cr13/14Cr17Ni2
   - **Fe-7** = 铁素体不锈钢（分7-1/7-2并列）：06Cr13Al/10Cr17
   - **Fe-8** = 奥氏体不锈钢（**分8-1不含Mo / 8-2含Mo，并列**）：304系→8-1，316系→8-2
   - **Fe-5(A/B/C) = Cr-Mo 耐热钢，不是不锈钢**（与Fe-6/7/8严格分离）

2. **母材覆盖规则修正**（关键业务逻辑）：
   - 不同类别的组别覆盖策略不同，数据驱动配置：
     - `progressive`（递进，高覆盖低）：Fe-1/Fe-2/Fe-3/Fe-4（按强度）
     - `parallel`（并列，互不覆盖）：Fe-7/Fe-8（按成分，如Fe-8-1与8-2互不覆盖）
   - 修正了阶段0"所有类别高组别覆盖低组别"的错误（Fe-8-1与8-2是并列而非递进）

3. **表7冲击约束已实现**（2023/2011一致）：
   - 当 **1.5 ≤ t ≤ 10mm 且有冲击试验要求**时，厚度上限由 2t 收紧为 **t+1.5mm**

4. **2023版电源类型升级**：全焊接方法（SMAW/GTAW/SAW/GMAW/PAW/EGW）电源类型均升级为重要因素，已全覆盖。

## 五、概念澄清（已落实，避免建模混淆）

| 概念 | 所属体系 | weldAI 采用 |
|------|---------|------------|
| F-Number | ASME IX QW-432（焊工技能相关） | 不直接用 |
| A-Number | ASME IX QW-443（熔敷金属成分） | 不直接用 |
| **表2/3/4 分类栏位** | **NB/T 47014** | ✅ **变更判定主键** |
| FXXAX 组合型号 | GB/T 5293/12470（埋弧焊焊丝-焊剂） | 型号解析器处理，≠ ASME F-No |

**核心原则**：严格按 NB/T 47014 建模，以**型号**（国标统一规格）为唯一可信字段，
**牌号**（焊材厂商品名）仅作人工录入辅助。跨标准工程需另维护映射表。
